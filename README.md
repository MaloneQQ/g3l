# Installing

```
npm i g3l -g
```

#  Usage: g3l [options]

  Options:

    -h, --help                 output usage information
    -m, --message <message>    Commit message
    -b, --new_branch <branch>  Git push origin as a new branch
    -p, --publish              After process publish repo as a npm package
    -i, --init                 Git init
    -v, --verbose              Show process
